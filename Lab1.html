We are doing so utilizing the cumulative distrbution function (CDF). Looking at a typical bell curve graph we can see that the data spread is split equally between the mean. That is further confirmed with the first value which asks what is the probablity that a value is between negative infinity and the mean plus zero standard deviation (which is just the mean). It's 50%. Also we can further validate this claim since we know that within one standard deviation of the mean, it encapsulates 68% of the data. Using this knowledge we can say that the probablity that a value is between negative infinity and two standard deviations above the mean is (1/2)(.95) + (.50) = 97.5% which is pretty close to the value we calculated above which was 97.72%.

Problem 1.2.A: An exponential distribution is a continuous "probablity distribution of the time between events in a Poisson point process" - which is when events happen independently and continously at an average constant rate. The probablity density function is f(x) = lambdae^(-lambda x) where lambda is considered the rate and must be larger than zero. https://en.wikipedia.org/wiki/Exponential_distribution

Problem 1.2.B: Both graphs are showcasing the exponential probablity distribution with 500 samples spaced evenly between zero and ten. In the blue is a 100k randomly generated points that mimics the exponential curve. When plotting the exponential distrbution, I chose lambda to be equal to one. Thus the PDF = e^(-x). This function also has several properties such as the expected value = 1/lambda and the varience = 1/(lambda)^2 = mean. Therefore the mean of our data is 1/1 = 1. Also the median is given by ln(2)/lambda < E[X]. That means that from zero to ln(2)/1 = .693, contains 50% of the values of the exponential distrubution.

However, the there's a difference between the top and bottom graph in that the bottom one is a log scale graph. Changing the y axis into a logarithmic scale allows the viewer to visualize a large range of values in a very compact way - giving a better insight on the pattern of the data and also allowing a visual verification of the exponential law of the function.

Problem 2.1.A: A binomial distribution is a "discrete probablity distribution of the number of successes in a sequence of n independent experiments, each asking a yes-no question, and each with its own Boolean-valued outcome." The probability mass function (PMF) is [n(factorial)] / [k(factorial)(n-k)(factorial)] p^k * (1-p)^(n-k). Where p is the probablity of success and n is the amount of trials. https://en.wikipedia.org/wiki/Binomial_distribution

Above shows with n trials and the probablity of success being 50% what is the spread across those trials. You can see since it's a 50/50 shot, the spread is mirrored across the medium which is 5.
